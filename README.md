## Установка и сборка
- `npm install` — установка пакетов,
- `npm run lint` - запуск StyleLint и ESLint с автофиксом,
- `npm run dev` - запуск в режиме разработки,
- `npm run build` — сборка и запуск проекта на 3000 порту,
- `npm run test` - запуск тестов.
